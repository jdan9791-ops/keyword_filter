## ✅ 핵심 도메인 지식 (조팀장님께 직접 배운 내용)

### 경쟁 법률사무소 블로그 운영 방식 (2026-04-16 확인)
- 법률사무소/법무법인들은 **수십 개의 블로그 계정**을 운영함
- 각 블로그마다 매일 사기 피해 정보 글을 대량 포스팅 (SEO 마케팅 전략)
- 목적: 피해자가 사기 업체명 검색 → 자기 블로그 노출 → 수임 유도
- 모든 글 본문에 **자기 법인명이 반드시 포함**됨
- 따라서 법인명으로 네이버 블로그 검색하면 해당 법인의 **모든 블로그 글이 수집됨**

### 블로그 ID 패턴
- `fudge14634`, `ivorydesire89`, `law897_`, `castle_h1120` 같은 랜덤/의미없어 보이는 ID → **전부 법률사무소 공식 마케팅 블로그**
- 제목 끝에 **(사칭)** 패턴 → 법률사무소 블로그의 전형적인 포스팅 형식
- 이를 "스팸"으로 오판하면 안 됨 ← **과거 실수**

- 네이버 블로그 API → 법인명 검색 → 날짜 필터 → **전부 수집**
